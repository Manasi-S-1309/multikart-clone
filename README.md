# Multikart Homepage Clone

A static clone of the **Multikart e-commerce homepage**, built using **HTML, CSS, and JavaScript**.  
This project was created to practice responsive design, modern UI/UX principles, and showcase frontend development skills.

---

## Live Demo
Check out the project here: [Live Demo](https://manasi-s-1309.github.io/multikart-clone/)

---

## Features
- Responsive design for desktop and mobile
- Clean and modern UI layout
- Product grid, banners, and navigation bar
- Organized file structure for scalability
- Cross-browser compatibility

---

## Tech Stack
- **HTML5** – Structure
- **CSS3** – Styling, Flexbox, Grid, Media Queries
- **JavaScript (ES6)** – Interactivity
- **Git/GitHub** – Version control and hosting

---

## Learning & Challenges
- Learned how to structure complex layouts with Flexbox and Grid
- Faced challenges with cross-browser testing and responsiveness
- Improved skills in clean coding and modular CSS
